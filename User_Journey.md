# User Journey

1. User opens the app.

↓

2. Registers using Mobile Number and OTP.

↓

3. Completes Business Profile.

↓

4. Uploads Documents.

↓

5. AI verifies documents.

↓

6. AI checks loan eligibility.

↓

7. User selects loan offer.

↓

8. Loan application submitted.

↓

9. Credit Officer reviews application.

↓

10. Loan approved and amount credited.

↓

11. User tracks EMI and repayment.
